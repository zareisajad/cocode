# cocode
coCode is a open source platform to help self learners in programming world to stay consistent, motivate each other and grow through collaborative micro-community environment.

### Watch me build cocode:
- [Youtube playlist](https://www.youtube.com/playlist?list=PLi6rlPMo5ZJCI_xmO_ZjaGPKIRMFcHR9G)

## Who is coCode for? (target audience)

self learners in any programming field. beginners to experts who:

- feel isolated learning to code
- lack motivation and consistency
- want small communities, accountability
- need  a place to find people at their level

---
## Tech:

- **Frontend:** Next.js, Tailwind
- **Backend:** django + django rest framework 
- **DB:** Postgres 
- **Notes / diagrams:** obsidian, excalidraw
